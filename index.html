Terminal commands - Cheatsheet

  1. Print Working Directory

    Command: pwd

    Funktion: Gibt das aktuelle Arbeitsverzeichnis aus.

  2. Listing Directory

    Command: ls

    Funktion: Gibt den Inhalt des aktuellen Verzeichnises aus.

    • Zusatz: -1

      Command: ls -1

      Funktion: Gibt den Inhalt des aktuellen Verzeichnises in nur einer Spalte aus.

    • Zusatz: -l

      Command: ls -l

      Funktion: Gibt den Inhalt des aktuellen Verzeichnises in einem breiten Format aus.

    • Zusatz: -a

      Command: ls -a

      Funktion: Gibt den Inhalt des aktuellen Verzeichnises inklusive versteckter Dateien aus.

  3. Clear

    Command: clear

    Funktion: Leert den Terminal-Bildschirm von allen vorherigen Befehle

  4. Change Directory

    Command: cd

    Funktion: Änderung des Arbeitsverzeichnis.

    • Zusatz: ..

      Command: cd ..

      Funktion: Gehe zum übergeodneten Verzeichnis.

    • Zusatz: ~

      Command: cd ~

      Funktion: Gehe zum Benutzer-Verzeichnis.

  5. Make Directory

    Command: mkdir

    Funktion: Erstelle ein neues Verzeichnis.

  6. Touch

    Command: touch

    Funktion: Erstelle eine Datei.

  7. Move

    Command: mv

    Funktion: Verschiebt eine Datei.

  8. Copy

    Command: cp

    Funktion: Kopiert eine Datei.

  9. Remove

    Command: rm

    Funktion: Löscht eine Datei.

    • Zusatz: -r

      Command: rm -r

      Funktion: Löscht ein Verzeichnis und all seine Inhalte.

  10. Remove Directory

    Command: rmdir

    Funktion: Löscht ein leeres Verzeichnis.

  11. Manual

    Command: man

    Funktion: Öffnet ein Handbuch zu Befehlen.

  12. Calendar

    Command: cal

    Funktion: Zeigt einen Kalender für den aktuellen Monat, der aktuelle Tag ist dabei markiert.
