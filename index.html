Terminal Commands

  1. Print Working Directory

    Command: pwd

    What it does: Gibt das aktuelle Arbeitsverzeichnis aus.

  2. Listing Directory

    Command: ls

    What it does: Gibt den Inhalt des aktuellen Verzeichnises aus.

    • Zusatz: -1

      Command: ls -1

      What it does: Gibt den Inhalt des aktuellen Verzeichnises in nur einer Spalte aus.

    • Zusatz: -l

      Command: ls -l

      What it does: Gibt den Inhalt des aktuellen Verzeichnises in einem breiten Format aus.

    • Zusatz: -a

      Command: ls -a

      What it does: Gibt den Inhalt des aktuellen Verzeichnises inklusive versteckter Dateien aus.

  3. Clear

    Command: clear

    What it does: Leert den Terminal-Bildschirm von allen vorherigen Befehle

  4. Change Directory

    Command: cd

    What it does: Änderung des Arbeitsverzeichnis.

    • Zusatz: ..

      Command: cd ..

      What it does: Gehe zum übergeodneten Verzeichnis.

    • Zusatz: ~

      Command: cd ~

      What it does: Gehe zum Benutzer-Verzeichnis.

  5. Make Directory

    Command: mkdir

    What it does: Erstelle ein neues Verzeichnis.

  6. Touch

    Command: touch

    What it does: Erstelle eine Datei.

  7. Move

    Command: mv

    What it does: Verschiebt eine Datei.

  8. Copy

    Command: cp

    What it does: Kopiert eine Datei.

  9. Remove

    Command: rm

    What it does: Löscht eine Datei.

    • Zusatz: -r

      Command: rm -r

      What it does: Löscht ein Verzeichnis und all seine Inhalte.

  10. Remove Directory

    Command: rmdir

    What it does: Löscht ein leeres Verzeichnis.

  11. Manual

    Command: man

    What it does: Öffnet ein Handbuch zu Befehlen.

  12. Calendar

    Command: cal

    What it does: Zeigt einen Kalender für den aktuellen Monat, der aktuelle Tag ist dabei markiert.

+ -

Git Commands

  1. Initialize

    Command: git init

    What it does: Initializes a new local repository.

  2. Add

    Command: git add

    What it does: Adds everything to the local repository.

  3. Commit

    Command: git commit

    What it does: Records the file permanently in the version history.

    • Zusatz: -m "comment here"

      Command: git commit -m "comment here"

      What it does: Records the file permanently in the version history with a discriptive comment.

  4. Status

    Command: git status

    What it does: Lists all the files that have to be added/committed.

  5. Differnce

    Command: git diff

    What it does: Shows the file differences which are not yet staged.

  6. Log

    Command: git log

    What it does: List the version history for the current branch.

    • Zusatz: --oneline

      Command: git log --oneline

      What it does: List the version history for the current branch in a more condensed way.

  7. Branch

    Command: git branch

    What it does: Lists all the local branches in the current repository.

    • Zusatz: -a

      Command: git branch -a

      What it does: Lists all the local branches.

    • Zusatz: "branch_name"

      Command: git branch "branch_name"

      What it does: Creates a new branch.

    • Zusatz: -d "branch_name"

      Command: git branch -d "branch_name"

      What it does: Deletes the branch.

  8. Checkout

    Command: git checkout "branch_name"

    What it does: Switch from one branch to another.

  9. Merge

    Command: git merge

    What it does: Merges the specified branch’s history into the current branch.

  10. Remote

    Command: git remote

    What it does: Connect the local repository to the remote server.

  11. Push

    Command: git push

    What it does: Sends the committed changes to your remote repository.

  12. Pull

    Command: git pull

    What it does: Fetches and merges changes on the remote server to your working directory.
